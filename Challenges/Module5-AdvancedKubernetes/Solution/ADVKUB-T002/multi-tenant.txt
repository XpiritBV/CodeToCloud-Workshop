* Copy the values.yml and change the name to web-customerx and run on port 8080
* Deploy the helm chart with different values
```
helm install web-customerx .\content-web\ -f .\customerx-values-contentweb.yaml
```
* Update the content-ingress file to have customerx as a new path
```
          - path: /customerx/(.*)
            backend:
              serviceName: web-customerx
              servicePort: 8080      
```
* Apply the new content-ingress
```
kubectl apply -f content-ingress.yml
